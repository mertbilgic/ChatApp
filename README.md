# ChatApp

**Dependencies:** Python 3.8.0, Pip, Virtualenv, Npm

### Create virtual environment 

```sh
git clone https://github.com/mertbilgic/ChatApp
cd ChatApp
virtualenv venv
```

### Install depedencies

```sh
$ source venv/bin/activate
$ pip install -r requirements.txt
```

### Install Styles and Scripts

```sh
cd static
$ npm install
```
### Start

```sh
$ python app.py 
```
